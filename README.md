# Omega_AI Bio Link

Красивая landing page (bio link) для Omega_AI с ссылками на YouTube и Instagram.

## Особенности

- 🎨 Красивый дизайн с glass-морфизмом
- ✨ Плавные анимации и переходы
- 📱 Полная адаптивность для всех устройств
- 🔗 Умное открытие приложений (YouTube, Instagram) или браузера
- 🎯 Быстрая загрузка с использованием Tailwind CSS

## Структура проекта

```
├── omega.html              # Главная страница
├── logo1.jpg              # Логотип/аватар
├── favicon-*.png          # Favicon файлы
├── android-chrome-*.png   # Иконки для Android
├── apple-touch-icon.png   # Иконка для iOS
├── favicon.ico            # ICO файл для совместимости
├── vercel.json            # Конфиг для Vercel
└── .vercelignore          # Файлы для игнорирования при деплое
```

## Быстрый старт

1. Клонируйте репозиторий
2. Откройте `omega.html` в браузере

## Деплой на Vercel

### Способ 1: С помощью GitHub

1. Создайте GitHub репозиторий и залейте код
2. Перейдите на [vercel.com](https://vercel.com)
3. Нажмите "New Project"
4. Импортируйте ваш GitHub репозиторий
5. Vercel автоматически развернет ваш проект

### Способ 2: С помощью Vercel CLI

```bash
npm install -g vercel
vercel
```
